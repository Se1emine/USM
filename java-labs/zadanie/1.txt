Лабораторная работа №1 – Классы

Цель – изучение среды разработки приложений. Создание класса Java по аналогии с классами в С++.

Создать класс Matrix, содержащий двумерный массив n x m целого типа, организовать ввод-вывод массива, поиск минимального и максимального элементов. Для ввода использовать класс Scanner, а для вывода метод System.out.println().
